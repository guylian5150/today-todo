# Today Todo

A focused, mobile-friendly Todo application built with Next.js and TypeScript.

## Features

- Add tasks
- Mark tasks as complete
- Delete tasks
- Keep tasks after a page reload with LocalStorage

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
pnpm lint
pnpm build
```
